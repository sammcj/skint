# Task Completion Checklist

When completing a task on this project:

1. **Lint**: `make lint` - ensure golangci-lint passes
2. **Format**: `make fmt` - ensure code is formatted
3. **Test**: `make test` - run all tests, verify no regressions
4. **Build**: `make build` - verify binary builds successfully
5. **No debug statements** left in code
6. **Error handling** in place for all new code paths
7. **Australian English** spelling in comments and docs
